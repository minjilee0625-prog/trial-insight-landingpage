# Product

## Register

brand

## Users

의뢰사(Sponsor) 담당자 — 제약사 또는 CRO에서 임상시험을 발주·관리하는 실무 담당자. 여러 기관·과제를 동시에 관리하며, 신뢰할 수 있는 데이터 품질과 빠른 이슈 감지를 요구한다. 임상연구 규제 환경에 민감하고, 검증된 도구만을 채택하려는 신중한 구매 의사결정자다.

## Product Purpose

Trial Insight는 임상연구 모니터링 특화 플랫폼이다. EDC 데이터 자동 동기화, AI 기반 이상 징후 감지, 뉴스레터·리포트 자동 생성 등을 통해 CRA의 반복 수작업을 자동화하고, 의뢰사에게는 실시간 가시성을 제공한다. 성공의 기준은 방문한 Sponsor 담당자가 "이 팀은 임상 현장을 안다"는 신뢰를 갖고 데모 신청으로 전환하는 것이다.

## Brand Personality

신뢰 · 정밀 · 전문성 — 의료/규제 환경에 걸맞은 권위 있는 목소리, 기술적 정확성을 드러내는 데이터 중심 표현, 복잡함을 단순하게 만드는 전문가적 자신감.

## Anti-references

특별히 지정된 안티 레퍼런스 없음. 단, 다음 두 방향은 브랜드와 충돌한다:
- 과도하게 화려한 SaaS 스타트업 스타일 (그라디언트 남발, 애니메이션 과다)
- 구닥다리 병원 소프트웨어 느낌 (딱딱하고 무거운 의료기기 UI)

## Design Principles

1. **현장에서 검증된 신뢰** — 실제 임상 현장 경험을 근거로 말한다. 수치, 사례, 구체적 기능이 추상적 주장을 대신한다.
2. **복잡함을 정제된 명확함으로** — 임상연구의 복잡성을 단순하게 보여주되, 단순화를 위해 정확성을 희생하지 않는다.
3. **전문가에게 말을 걸다** — UI 카피와 설명이 임상연구 문법을 사용한다. 일반 SaaS 마케팅 언어를 피한다.
4. **데이터가 주인공** — 기능 설명보다 실제 데이터 화면과 결과물이 앞에 온다.
5. **신뢰는 세부에서 만들어진다** — 여백, 타이포그래피, 컴포넌트의 정밀함이 제품 품질의 신호다.

## Accessibility & Inclusion

- WCAG 2.1 AA 기준 (한국어 서비스 기본)
- 주요 사용자가 데스크탑 업무 환경이므로 데스크탑 퍼스트, 모바일 반응형 지원
- 색상 대비 최소 4.5:1 (본문 텍스트), 3:1 (대형 텍스트)
